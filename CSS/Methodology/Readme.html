Методологии

Методологии

В CSSSR используются две основные методологии:

  • BEM
  • CSS Modules

Требования к методологиям

  • Компонентный подход — верстка должна разделяться на отдельные независимые компоненты
  • Изолированность — каждый компонент должен быть изолирован от других компонентов. Компонент не должен влиять на дочерние компоненты и, наоборот, компонент не должен наследовать стили от родителя.
  • Изменяемость — каждый компонент должен быть настраиваемым. Компонент должен настраиваться явным образом и через какой-либо публичный интерфейс, вид компонента не должен зависеть от его окружения.
